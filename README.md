## ❗️ Troubleshooting
| Problem                  | Solution                                      |
|--------------------------|-----------------------------------------------|
| Game doesn't start       | Run as Administrator                          |
| Low performance          | Lower resolution or graphics settings         |
| Black screen             | Switch to Borderless Windowed mode            |
| Antivirus warning        | Add the folder to exclusions                  |
| Controller issues        | Check controller settings in game             |
